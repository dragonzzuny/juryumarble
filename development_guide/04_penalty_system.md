# 주류마블 안드로이드 앱: 벌칙 시스템 상세 정의서

## 1. 개요

이 문서는 '주류마블' 앱의 핵심 재미 요소이자 책임 있는 음주 문화를 유도하는 장치인 **벌칙 시스템**을 상세하게 정의합니다. 벌칙은 단순히 술을 마시게 하는 것을 넘어, 게임의 상호작용을 증진시키고 다양한 재미를 부여하는 역할을 합니다. 본 문서는 벌칙의 종류, 강도, 적용 방식 및 안전장치에 대한 명확한 기준을 제시합니다.

## 2. 벌칙 시스템 설계 철학

- **재미와 다양성 (Fun & Variety)**: 마시기만 하는 단순한 벌칙에서 벗어나, 행동, 미션, 룰 변경 등 다양한 종류의 벌칙을 제공하여 게임의 지루함을 없애고 예측 불가능한 재미를 선사합니다.
- **공정성과 선택권 (Fairness & Choice)**: 벌칙의 강도는 사용자가 사전에 설정한 값(주량, 수위)에 따라 조절되며, 플레이어는 경우에 따라 벌칙을 피하거나 다른 플레이어에게 넘길 수 있는 선택권을 가집니다.
- **책임 있는 음주 (Responsible Drinking)**: 과음을 방지하기 위한 안전장치를 시스템 곳곳에 내장합니다. 휴식 유도, 논알콜 옵션, 강요 금지 문화 조성을 최우선으로 고려합니다.

## 3. 벌칙의 분류 (Penalty Categories)

벌칙은 크게 5가지 카테고리로 분류되어, 다양한 카드와 보드 칸 이벤트를 통해 발동됩니다.

| 카테고리 | 설명 | 주요 목표 | 예시 |
|---|---|---|---| 
| **마시기 (Drinking)** | 가장 기본적인 벌칙으로, 지정된 양의 술을 마십니다. | 게임의 기본 진행 | 원샷, 반샷, 러브샷, 다같이 마시기 |
| **행동/미션 (Action/Mission)** | 신체적 행동, 연기, 퀴즈 등 특정 미션을 수행합니다. 실패 시 대체 벌칙(주로 마시기)을 받습니다. | 상호작용 증진, 아이스브레이킹 | 노래 부르기, 특정 단어 금지, 밸런스 게임, 진실 혹은 도전 |
| **룰 변경 (Rule Change)** | 일정 시간 또는 특정 조건 동안 모든 플레이어에게 적용되는 새로운 규칙을 추가합니다. | 게임의 역동성 부여 | '나' 금지어, 특정 손 모양 유지하기, 영어만 쓰기 |
| **상태 이상 (Status Effect)** | 특정 플레이어에게 일시적인 페널티나 버프를 부여합니다. | 전략적 요소 추가 | 다음 턴 한 번 쉬기, 주사위 1개만 굴리기, 벌칙 면제권 획득 |
| **상호작용 (Interaction)** | 다른 플레이어와 직접적인 상호작용을 유도하는 벌칙입니다. | 관계 증진, 협력/경쟁 유도 | 왼쪽 사람과 러브샷, 오른쪽 사람에게 술 따라주기, 한 명 지목해서 대신 마시게 하기 |

## 4. 벌칙 강도 스케일링 시스템

벌칙의 강도는 게임의 공정성과 재미를 위해 여러 변수에 의해 동적으로 조절됩니다. 이는 `scalePenalty` 알고리즘을 통해 구현됩니다. (알고리즘 설계서 참조)

- **기본 강도 (Base Scale)**: 모든 벌칙 카드는 `penaltyScale` (e.g., 1.0)이라는 기본 강도를 가집니다. 이는 '소주 1샷'에 해당하는 기준값입니다.
- **수위 필터 (Severity Filter)**: 사용자가 게임 시작 시 설정한 '순한맛/보통/매운맛'에 따라 특정 수위 이상의 카드는 덱에서 원천적으로 제외됩니다.
- **주량 레벨 (Tolerance Level)**: 플레이어가 설정한 '라이트/노말/하드'에 따라 최종 벌칙의 양이 조절됩니다.
    - **라이트**: 기본 벌칙량의 50% 적용 (e.g., 1샷 → 반샷 또는 1모금)
    - **노말**: 기본 벌칙량 100% 적용
    - **하드**: 기본 벌칙량 150% 적용 (e.g., 1샷 → 1.5샷)
- **주종/단위 변환 (Unit Conversion)**: 설정된 주종과 단위('샷', '잔', '모금')에 따라 최종 벌칙 텍스트가 변환됩니다. 예를 들어, '맥주'와 '모금'으로 설정 시, 1샷은 '3모금'과 같이 내부적으로 정의된 규칙에 따라 변환됩니다.

**계산 공식 예시:**
> `최종 벌칙량 = ceil(카드 기본 강도 × 주량 레벨 배율)`

## 5. 벌칙 상세 정의 (예시)

다음은 각 카테고리별 벌칙의 구체적인 예시입니다.

| 카테고리 | 벌칙 이름 | 수위 | 내용 | 실패 시 대체 벌칙 |
|---|---|---|---|---|
| **마시기** | 원샷 웨이브 | 보통 | 현재 플레이어부터 시계방향으로 한 명씩 원샷. | (없음) |
| | 흑기사/흑장미 | 보통 | 지목한 이성이 대신 마셔주기. 소원 들어주기. | 지목 거부 시 본인 2배 마시기 |
| | 의리 게임 | 매운맛 | 술잔을 채우며 한 명씩 "의리!" 외치기. 넘치는 사람이 마시기. | (없음) |
| **행동/미션** | 밸런스 게임 | 순한맛 | 제시된 두 가지 선택지 중 하나를 고르고 이유를 설명해야 함. | 1샷 마시기 |
| | 초성 퀴즈 | 순한맛 | 제시된 초성의 단어를 가장 늦게 맞추는 사람 마시기. | (없음) |
| | 폰 공개 | 매운맛 | 1분 동안 무작위로 한 명에게 휴대폰 모든 기능(갤러리, 카톡 등) 공개. | 3샷 마시기 |
| **룰 변경** | 금지어 게임 | 보통 | 다음 턴까지 특정 단어(e.g., "나", "너", "진짜")를 말하면 마시기. | (없음) |
| | 바보 게임 | 보통 | 다음 턴까지 질문에 무조건 동문서답해야 함. 실패 시 마시기. | (없음) |
| **상태 이상** | 벌칙 면제권 | 순한맛 | 이 카드를 보유한 플레이어는 다음 벌칙을 1회 면제받을 수 있음. | (없음) |
| | 무인도 | 보통 | 다음 턴을 한 번 건너뜀. | (없음) |
| **상호작용** | 러브샷 | 보통 | 지목한 사람과 러브샷. | 지목 거부 시 본인 2배 마시기 |
| | 대신 마시기 | 순한맛 | 오른쪽 사람이 대신 1샷 마셔주기. | (없음) |

## 6. 과음 방지 및 안전장치 (Safety Mechanisms)

책임 있는 음주 문화를 장려하고 사용자를 보호하기 위해 다음과 같은 안전장치를 필수적으로 구현합니다.

- **연속 벌칙 완화**: 한 플레이어가 연속으로 3회 이상 벌칙에 걸릴 경우, 4번째 벌칙부터는 강도가 자동으로 50% 감소하거나 '물 마시기' 카드로 대체됩니다.
- **휴식 권고 시스템**: 게임 시작 후 일정 시간(e.g., 45분)이 경과하거나, 한 플레이어가 누적 벌칙량을 초과할 경우 "잠시 쉬는 시간을 갖는 건 어떨까요?" 와 같은 휴식 권고 팝업이 표시됩니다.
- **'물 마시기' 카드**: 카드 덱에 일정 비율(e.g., 5%)의 '물 한 컵 마시기' 카드를 포함하여 자연스러운 수분 섭취를 유도합니다.
- **항복(기권) 기능**: 플레이어는 언제든지 '항복' 버튼을 통해 게임에서 자발적으로 나갈 수 있습니다. 항복한 플레이어는 더 이상 벌칙을 받지 않으며, 이는 패배나 약함이 아닌 '책임감 있는 선택'으로 표현됩니다.
- **논알콜 모드**: 게임 시작 시 '논알콜'을 주종으로 선택하면 모든 '마시기' 벌칙이 '물 마시기' 또는 가벼운 미션으로 자동 대체됩니다.
