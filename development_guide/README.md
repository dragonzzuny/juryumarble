# 주류마블: 파티 보드게임 앱 - 통합 개발 가이드

## 1. 프로젝트 개요

본 문서는 안드로이드 스튜디오 기반의 '주류마블: 파티 보드게임 앱' 개발을 위한 통합 가이드입니다. 이 프로젝트는 술자리에서 별도의 준비물 없이 스마트폰 하나로 즐길 수 있는 부루마블 형식의 파티 게임을 제공하는 것을 목표로 합니다.

이 가이드는 기획 단계의 요구사항 분석부터 데이터베이스 설계, 시스템 아키텍처, 핵심 알고리즘, 그리고 게임의 핵심 콘텐츠인 카드 시스템에 이르기까지 개발에 필요한 모든 정보를 상세하고 체계적으로 담고 있습니다. 개발팀은 이 문서를 통해 프로젝트의 전체적인 그림을 이해하고, 일관된 방향성을 유지하며 협업할 수 있습니다.

## 2. 문서 구성

본 통합 개발 가이드는 다음과 같은 상세 문서들로 구성되어 있습니다. 각 문서는 특정 주제에 대해 심도 있는 정보를 제공하므로, 개발 진행 시 해당 파트를 참고하시기 바랍니다.

- **[01_data_schema.md](./01_data_schema.md)**
    - **내용**: 앱에서 사용되는 모든 데이터의 구조를 정의하는 데이터베이스 스키마 명세서입니다. Android Room을 기반으로 한 Entity 클래스, 테이블 간의 관계(ERD), 각 컬럼의 속성 등을 상세히 기술합니다.

- **[02_architecture_and_plan.md](./02_architecture_and_plan.md)**
    - **내용**: 프로젝트의 기술적 기반이 되는 시스템 아키텍처와 MVP 개발 계획을 다룹니다. 클린 아키텍처(Clean Architecture)의 각 레이어(Presentation, Domain, Data)의 역할과 책임, 권장 기술 스택, 그리고 8주간의 상세 개발 마일스톤을 포함합니다.

- **[03_algorithm_and_logic.md](./03_algorithm_and_logic.md)**
    - **내용**: 게임의 핵심 두뇌인 게임 엔진의 작동 원리를 설명합니다. 게임의 흐름을 제어하는 상태 머신(State Machine), 주사위 굴리기, 카드 뽑기, 벌칙 강도 조절 등 구체적인 알고리즘과 로직을 유사 코드(Pseudo-code)와 함께 정의합니다.

- **[04_penalty_system.md](./04_penalty_system.md)**
    - **내용**: 게임의 재미와 균형을 책임지는 벌칙 시스템을 상세히 정의합니다. 벌칙의 종류, 강도, 동적 스케일링 방식과 더불어, 책임 있는 음주 문화를 장려하기 위한 과음 방지 안전장치에 대해 기술합니다.

- **[05_card_content.md](./05_card_content.md)**
    - **내용**: 게임의 가장 중요한 콘텐츠인 카드 덱의 전체 목록입니다. MVP에 포함될 '기본 카드팩' 200종에 대한 상세 정보(ID, 타입, 수위, 내용 등)를 포함하고 있어, 기획자와 개발자 간의 명확한 소통을 돕습니다.

## 3. 개발 시작 가이드

1.  **문서 정독**: 개발에 참여하기 전, 본 가이드의 모든 문서를 정독하여 프로젝트의 목표와 기술적 설계를 완전히 숙지합니다.
2.  **환경 설정**: `02_architecture_and_plan.md` 문서에 명시된 기술 스택에 따라 개발 환경을 설정합니다.
3.  **프로젝트 구조 생성**: 제안된 모듈 및 패키지 구조에 따라 안드로이드 스튜디오 프로젝트를 생성합니다.
4.  **개발 계획 준수**: 마일스톤에 따라 개발을 진행하며, 각 단계의 완료 기준(Definition of Done)을 충족하는지 지속적으로 확인합니다.

본 가이드가 성공적인 프로젝트 완료에 훌륭한 길잡이가 되기를 바랍니다.
